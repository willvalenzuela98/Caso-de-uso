Caso de Uso - Reserva de Vuelo

1. Título del Caso de Uso

Reserva de Vuelo

2. Actor(es)

Cliente

3. Descripción

Permite al Cliente buscar y reservar vuelos.

4. Precondiciones

El Cliente debe tener una cuenta en el sistema.

5. Flujo Principal de Eventos

  1. El Cliente inicia sesión en el sistema.
  2. El Cliente busca vuelos según sus preferencias (fecha, destino, etc.).
  3. El sistema muestra los vuelos disponibles.
  4. El Cliente selecciona un vuelo y proporciona los datos necesarios.
  5. El Cliente confirma la reserva y realiza el pago.
  6. El sistema envía la confirmación de la reserva al Cliente.

6. Postcondiciones

La reserva se registra correctamente en el sistema.

7. Excepciones

Si el pago falla, se notifica al Cliente y la reserva no se realiza.
